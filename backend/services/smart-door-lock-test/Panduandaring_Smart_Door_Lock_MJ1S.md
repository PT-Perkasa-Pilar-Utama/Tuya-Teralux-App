## Petunjuk Penggunaan

### Informasi Dasar

1. **Kata sandi default** adalah `123456`
2. **Tombol #** berfungsi untuk konfirmasi atau masuk ke tombol fungsi menu
3. **Tombol \*** berfungsi untuk menghapus atau kembali ke menu sebelumnya
4. Tekan **\* + #** untuk masuk ke mode administrator
5. Setelah menambahkan kata sandi administrator, kata sandi `123456` akan terhapus secara otomatis
6. Tekan tombol **#** untuk membunyikan bel

---

## Pemasangan Baterai

1. Tekan tombol **Press** pada panel pintu bagian dalam untuk membuka kompartemen baterai
2. Pasang **4 baterai jenis AA** ke dalam kompartemen baterai dan pastikan polaritas baterai telah terpasang dengan benar
3. Pasang kembali penutup kompartemen baterai

---

## Menghubungkan Kabel

Hubungkan kabel pada pintu bagian luar dengan kabel daya pada pintu bagian dalam dengan cara sejajarkan alur, kemudian pintu akan secara otomatis menyala dan siap dioperasikan.

## Mengubah Arah Gagang Pintu

### 1) Mengubah Arah Gagang Pintu Bagian Luar

1. Posisikan pintu bagian luar secara terbalik seperti ilustrasi pada gambar
2. Tempatkan 1 baut ke dalam lubang yang ditunjuk arah panah
   - **Catatan:** Baut termasuk dalam kelengkapan aksesori yang Anda dapatkan
3. Kencangkan baut menggunakan obeng (obeng tidak termasuk dalam paket pembelian) hingga terpasang erat

## Pengoperasian Awal

1. Setelah pintu dihubungkan ke daya, tekan tombol **\* + #** untuk masuk ke mode manajemen
2. Masukkan kata sandi administrasi default yaitu `123456`, kemudian pintu akan masuk ke mode manajemen
3. Pada saat ini, segera daftarkan kata sandi administrasi

## Manajemen Administrator

### 1) Menambahkan Kata Sandi Administrator

1. Usap panel untuk menyalakan panel, lalu tekan tombol **\*** dan **#** untuk masuk ke mode manajemen
2. Masukkan kata sandi administrasi default yaitu `123456`
3. Tekan tombol **1** untuk masuk ke menu penambahan pengguna
4. Tekan tombol **1** untuk masuk ke mode administrator
5. Tekan tombol **1** lagi untuk masuk ke pengaturan kata sandi
6. Daftarkan kata sandi administrator dengan memasukkan **6 digit angka** (misalnya, `232323`), kemudian tekan tombol **#** untuk konfirmasi
7. Masukkan ulang kata sandi administrator tadi, kemudian tekan tombol **#** untuk konfirmasi
8. Prompt suara akan berbunyi **"Add successful"**

**Catatan:** Setelah mengganti kata sandi administrator, kata sandi bawaan `123456` akan terhapus secara otomatis.

### 2) Menambahkan Sidik Jari Administrator

1. Usap panel untuk menyalakan panel, lalu tekan tombol **\*** dan **#** untuk masuk ke mode manajemen
2. Masukkan kata sandi administrasi default yaitu `123456`
3. Tekan tombol **1** untuk masuk ke menu penambahan pengguna
4. Tekan tombol **1** untuk masuk ke mode administrator
5. Tekan tombol **2** untuk masuk ke pengaturan sidik jari
6. Tempelkan sidik jari sebanyak **7 kali** pada area lingkaran yang menyala di engsel pintu dan ikuti instruksi prompt suara
7. Setelah berhasil, prompt suara akan berbunyi **"Add successful"**

### 3) Menambahkan Kartu Administrator

1. Usap panel untuk menyalakan panel, lalu tekan tombol **\*** dan **#** untuk masuk ke mode manajemen
2. Masukkan kata sandi administrasi default yaitu `123456`
3. Tekan tombol **1** untuk masuk ke menu penambahan pengguna
4. Tekan tombol **1** untuk masuk ke mode administrator
5. Tekan tombol **3** untuk masuk ke pengaturan kartu
6. Tempelkan kartu pada area sensor kartu di panel
7. Setelah berhasil, prompt suara akan berbunyi **"Add successful"**

## Manajemen Pengguna Umum

### 1) Menambahkan Kata Sandi Pengguna Umum

1. Usap panel untuk menyalakan panel, lalu tekan tombol **\*** dan **#** untuk masuk ke mode manajemen
2. Masukkan kata sandi administrasi
3. Tekan tombol **1** untuk masuk ke menu penambahan pengguna
4. Tekan tombol **2** untuk masuk ke mode pengguna umum
5. Tekan tombol **1** untuk masuk ke pengaturan kata sandi
6. Masukkan **6 digit angka** untuk kata sandi pengguna, kemudian tekan **#** untuk konfirmasi
7. Masukkan ulang kata sandi, kemudian tekan **#** untuk konfirmasi
8. Prompt suara akan berbunyi **"Add successful"**

### 2) Menambahkan Sidik Jari Pengguna Umum

Ikuti langkah yang sama seperti menambahkan administrator, tetapi pilih menu **pengguna umum** (tombol 2) pada langkah ke-4.

### 3) Menambahkan Kartu Pengguna Umum

Ikuti langkah yang sama seperti menambahkan administrator, tetapi pilih menu **pengguna umum** (tombol 2) pada langkah ke-4.

### 4) Menghapus Pengguna

1. Usap panel untuk menyalakan panel, lalu tekan tombol **\*** dan **#** untuk masuk ke mode manajemen
2. Masukkan kata sandi administrasi
3. Tekan tombol **2** untuk masuk ke menu penghapusan pengguna
4. Pilih metode yang ingin dihapus:
   - **1** = Administrator
   - **2** = Pengguna umum
   - **3** = Semua pengguna
5. Ikuti instruksi prompt suara untuk menyelesaikan penghapusan

## Pengaturan Sistem

### 1) Pengaturan Volume

1. Masuk ke menu pengaturan sistem (tombol **3** dari menu utama)
2. Tekan tombol **1** untuk pengaturan volume
3. Pilih level volume:
   - **1** = Volume tinggi
   - **2** = Volume sedang
   - **3** = Volume rendah
   - **4** = Mode senyap

### 2) Pengaturan Bahasa

1. Masuk ke menu pengaturan sistem (tombol **3** dari menu utama)
2. Tekan tombol **2** untuk pengaturan bahasa
3. Pilih bahasa yang diinginkan

### 3) Pengaturan Waktu

1. Masuk ke menu pengaturan sistem (tombol **3** dari menu utama)
2. Tekan tombol **3** untuk pengaturan waktu
3. Masukkan tahun, bulan, tanggal, jam, dan menit sesuai format yang diminta

### 4) Pengaturan Jaringan (WiFi)

**PENTING:** Ini adalah langkah untuk menghubungkan door lock ke aplikasi Tuya/Smart Life.

1. Usap panel untuk menyalakan panel, lalu tekan tombol **\*** dan **#** untuk masuk ke mode manajemen
2. Masukkan kata sandi administrasi
3. Tekan tombol **3** untuk masuk ke menu pengaturan sistem
4. Tekan tombol **3** untuk masuk ke pengaturan fungsi
5. Tekan tombol **1** untuk masuk ke pengaturan jaringan
6. Prompt suara akan berbunyi **"Connecting gateway"** dan berbunyi **beep-beep** selama proses pencarian aplikasi

**Lanjut ke aplikasi Tuya/Smart Life:**

7. Buka aplikasi Tuya atau Smart Life di ponsel Anda
8. Pada halaman utama, tekan tombol **+** di pojok kanan atas
9. Klik **Add Device** untuk menambahkan device
10. Pastikan ponsel diletakkan sedekat mungkin dengan pintu
11. Tunggu beberapa saat, akan muncul perangkat dengan nama **U688S-WiFi-Pro**
12. Pilih perangkat tersebut untuk menghubungkan pintu ke aplikasi
13. Masukkan **WiFi name** dan **password** WiFi Anda, lalu klik **Next**
14. **PENTING:** Pastikan WiFi memiliki koneksi **2.4 GHz** (tidak support 5 GHz)
15. Tunggu beberapa saat hingga aplikasi berhasil menambahkan perangkat
16. Setelah berhasil, akan muncul indikator **"Added successfully"** di aplikasi

**Catatan Penting:**
- Aplikasi hanya dapat digunakan untuk menambahkan kata sandi sementara dan membuka kunci **hanya saat tombol bel ditekan**
- Jika bel tidak ditekan, pintu tidak dapat digunakan untuk membuka atau mengunci pintu dari jarak jauh
- Saat bel pada pintu dibunyikan atau pintu terbuka, notifikasi akan muncul di aplikasi

### 5) Reset Pabrik dari Menu

1. Masuk ke menu pengaturan sistem
2. Pilih opsi reset pabrik
3. Konfirmasi reset

## Reset Pabrik

### Reset Menggunakan Tombol Fisik

1. Terdapat **tombol reset** di bagian belakang panel pintu yang terletak di bagian dalam kompartemen baterai
2. Tekan dan tahan tombol reset tersebut selama **7 detik** hingga prompt suara berbunyi:
   - **"Restore factory settings, please confirm key"**
3. Tekan tombol **#** untuk konfirmasi
4. Prompt suara akan berbunyi **"Successful"** yang menandakan pengaturan telah direset ke pengaturan pabrik

## Suplai Daya Darurat

Saat daya baterai habis dan tidak terdapat kunci fisik, Anda dapat menggunakan suplai daya eksternal untuk membuka pintu:

1. Buka penutup di bagian bawah pintu bagian luar dengan cara mencungkilnya
2. Colok kabel **USB Type-C** ke port USB, kemudian hubungkan ujung kabel lainnya ke stopkontak menggunakan adaptor daya
3. Selanjutnya pintu akan langsung menyala

**Catatan:** Disarankan untuk menggunakan suplai daya dengan output arus **2A atau di atasnya**.

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| WiFi tidak terdeteksi | Pastikan menggunakan WiFi 2.4 GHz, bukan 5 GHz |
| Device tidak muncul di app | Pastikan pintu dalam mode pairing (beep-beep) dan ponsel dekat dengan pintu |
| Notifikasi tidak muncul | Cek pengaturan notifikasi di aplikasi |
| Baterai cepat habis | Gunakan baterai alkaline berkualitas tinggi |
